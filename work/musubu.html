mu/su/bu (2016)

Sculpture | 3D print, coloured thread


MuSuBu [to bind, to link, to tie]

A link is a kind of hybrid object, a para-object.

A chain for example, might be constructed from several interlocking metal rings, but to call these isolated rings ‘links’ would be a mistake. The link is the collaborative effect of intersecting parts; both a verb, and a noun.

A link has no dimension; it is simply an intersection between vectors.

When a knot is tied, the process takes place in time, no step can be skipped, and each must be reversed in the same order to be undone. The rope makes links to itself, but also to the knot it creates; a new object separate from it.

A link is a context, but also the result of that context.
